# AutismCare 🧩  
### A Unified Platform for Autism Spectrum Disorder Management

AutismCare is a web-based Final Year Project (FYP) designed to support **early screening, management, and monitoring of Autism Spectrum Disorder (ASD)** in young children. The system digitalizes standardized assessment workflows and enables structured collaboration between parents, clinicians, therapists, laboratories, and administrators.

---

## 📌 Project Overview

Early identification of autism is critical for timely intervention, yet existing processes are often manual, fragmented, and inaccessible—especially in underserved and rural areas. AutismCare addresses these challenges by providing a **centralized, role-based digital platform** that streamlines autism care from screening to therapy monitoring.

The platform integrates **ASQ-3** and **M-CHAT-R** screening tools, maintains secure child profiles, and facilitates coordinated decision-making among all stakeholders involved in autism care.

---

## 🎯 Objectives

- Enable **early screening** of Autism Spectrum Disorder  
- Digitalize standardized tools such as **ASQ-3** and **M-CHAT-R**  
- Provide a **unified platform** for parents, clinicians, therapists, and labs  
- Track child development, therapy plans, and progress reports  
- Improve accessibility to autism care in **rural and underserved areas**

---

## 🚀 Key Features

- 🧠 **Digital Autism Screening** (ASQ-3 & M-CHAT-R with automated scoring)
- 👶 **Child Profiles** with complete assessment history
- 👨‍⚕️ **Clinician Dashboard** for evaluation, notes, and referrals
- 🧑‍🏫 **Therapist Module** with therapy plan templates and progress tracking
- 🧪 **Laboratory Module** for test uploads and status updates
- 🔔 **Notification System** for appointments and updates
- 🔐 **Role-Based Authentication & Secure Data Handling**
- 🌐 **Progressive Web App (PWA)** for cross-device accessibility

---

## 🏗️ System Architecture

**Modules:**
- Parent / Caregiver  
- Clinician  
- Therapist  
- Laboratory / Audiometrist  
- Admin  

**Technology Stack:**
- **Frontend:** React.js, Tailwind CSS, shadcn/ui  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB / PostgreSQL  
- **Authentication:** JWT-based role authorization  
- **Deployment:** Web-based Progressive Web App (PWA)
